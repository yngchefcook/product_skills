# Пример: Information Architecture

## Вход (от пользователя)

> Небольшой сервис записи к мастерам (барбершоп): клиент выбирает мастера и время.

## Фрагмент IA

```markdown
## 2. Sitemap
барбершоп
├── Главная (витрина услуг + CTA «Записаться»)
├── Мастера
│   └── Профиль мастера (услуги, расписание)
├── Запись (выбор услуги → мастер → время → подтверждение)
├── Мои записи
└── Авторизация (вход / регистрация)

## 3. Глобальная навигация
| Позиция | Элемент | Куда ведёт | Приоритет |
| 1 | Записаться | Запись | Высокий |
| 2 | Мастера | Мастера | Средний |
| 3 | Мои записи | Мои записи | Средний |

## 4. Ключевой User Flow — «Первая запись»
Главная → Записаться → выбор услуги → выбор мастера → выбор времени → подтверждение
   ↓ нет свободного времени
предложить ближайшие слоты
```
